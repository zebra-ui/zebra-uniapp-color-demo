## 介绍

zebraColor是一款兼容全端的颜色选择器，通常用于自定义颜色的场景。

## 特性

- 全端兼容
- 微信小程序使用`wxs`，支付宝小程序使用`sjs`，滑动选择颜色时丝滑无卡顿。

## 安装

### npm方式

```bash
npm i @zebra-ui/color
```

```js
// pages.json

{
	"easycom": {
		"^zebra-(.*)": "@zebra-ui/color/components/zebra-$1/zebra-$1.vue"
	},
	"pages": [...],
	"globalStyle": {...}
}
```

### uni_modules方式

[插件市场](https://ext.dcloud.net.cn)直接导入即可
